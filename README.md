# Unity Pong Game

A Unity-based recreation of Pong built in C#. This project focuses on gameplay systems, collision handling, object movement, and real-time game logic.

## Features

- Classic Pong gameplay
- Paddle and ball movement
- Collision-based interactions
- Score tracking
- Real-time input handling
- Game state and round flow logic

## Tech Stack

- Unity
- C#
- 2D Physics
- State-Based Game Logic

## What I Built

- Implemented the core gameplay loop in C#
- Built collision-driven interactions between paddles, ball, and boundaries
- Designed score tracking and round reset behavior
- Debugged runtime issues affecting gameplay responsiveness and system consistency

## Core Systems

- Player input handling
- Ball movement and collision detection
- Score management
- Round reset and gameplay state flow

## How to Run

1. Clone the repository
2. Open the project in Unity
3. Load the main scene
4. Press Play in the Unity Editor

## Project Focus

This project was built to strengthen my understanding of:

- event-driven gameplay systems
- collision handling
- real-time game logic
- Unity scripting in C#
